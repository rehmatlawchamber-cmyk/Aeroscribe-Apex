# 📈 AeroScribe Apex V5.1
### **High-Conversion Sovereign Asset Generator**

AeroScribe Apex is a high-stakes AI copywriting engine designed to generate technical, industrial, and luxury product descriptions. It is mathematically optimized to evade AI detection (ZeroGPT) while maintaining a "Grit-Injected" human persona.

## 🏦 Strategic Architecture
- **Multi-Chamber Synthesis:** Generates high-impact copy in English, French, German, Italian, Spanish, and Arabic.
- **Surgical Evasion:** Uses asymmetric sentence structures and technical friction to achieve 100% human-grade scores.
- **Behavioral Targeting:** Includes psychological profiles such as *The Status Seeker*, *The Technical Skeptic*, and *The Paranoiac Investor*.
- **Auto-Select Logic:** AI-driven analysis of raw intelligence to determine the most devastatingly effective tone and audience.

## 🛠 Technical Stack
- **Language:** Python 3.9+
- **Interface:** Streamlit (Elite Dark Mode UI)
- **Engine:** Gemini 1.5 Flash API
- **Deployment:** Optimized for Streamlit Cloud / Railway

## 🚀 Deployment Instructions
1. Clone the repository to your local machine.
2. Install dependencies: `pip install -r requirements.txt`
3. Set your `GEMINI_API_KEY` in your environment variables/secrets.
4. Execute: `streamlit run app.py`
